# cs2-redie
**a plugin that allows players to go in Redie/Ghost mode**

<br>

## information:

### requirements
- [MetaMod](https://cs2.poggu.me/metamod/installation)
- [CounterStrikeSharp](https://github.com/roflmuffin/CounterStrikeSharp)

### commands
`css_redie` / `css_ghost`

`css_unredie` / `css_unghost`
